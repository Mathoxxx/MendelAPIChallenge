# TransactionsAPI

A basic Java 21, Spring Boot 3, and Maven project for the Mendel API Challenge.

## Requirements

- Java 21
- Maven 3.x

## Build and Run

To build the project:
```bash
mvn clean install
```

To run the application:
```bash
mvn spring-boot:run
```

The application will start on port `8080` by default.

## Testing

To run the test suite:
```bash
mvn test
```
